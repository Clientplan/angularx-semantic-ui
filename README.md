AngularX Components for <a href="http://semantic-ui.com/">Semantic UI</a>
<br>
`No jQuery Dependency`

# Install
```
npm install angularx-semantic-ui --save
import { L_SEMANTIC_UI_MODULE } from 'angularx-semantic-ui';
```
